# Features　特徴

「検索」、「タスクの編集」が出来ます。

# Requirement ライブラリ

* jquery 2.2.4

# Usage　使い方

タスクを自由に追加できます。
タスク左のチェックアイコンを押すことで、タスクの状態を管理できます。
タスク右のゴミ箱アイコンを押すと、ゆっくり消えていきます。
タスクをクリックすると、文字の編集ができます。
「フォーカスを外す」「エンターを2回押す」「シフトとエンターを同時に押す」と編集が終了し、タスクに反映されます。
検索欄で検索できます（前方一致）。
